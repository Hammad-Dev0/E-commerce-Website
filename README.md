## 🎯 Project Overview
The **Rania Collection** website is built to bridge the gap between traditional Pakistani craftsmanship and modern digital minimalism. As a web development student project, it emphasizes clean UI/UX, smooth scroll animations, and a seamless mobile-to-desktop transition.

- **Client:** Rania Collection (Bedsheets & Curtains)
- **Developer:** Hammad
- **Goal:** To create a high-converting, aesthetic landing page that connects users directly to the brand via WhatsApp.

---

## ✨ Aesthetic Philosophy
The design follows a **"Minimalist Maximalism"** style:
- **Typography:** Utilizing *Cormorant Garamond* for an editorial/luxury feel and *Montserrat* for modern readability.
- **Color Palette:** Dominated by `#fcfbf7` (Bone White) and `#8e8071` (Stone Gray) to evoke a sense of calm and premium quality.
- **Whitespace:** Strategic use of negative space to allow product imagery to breathe.

---

## 🚀 Core Features
- **Dynamic Scroll Reveals:** Elements float into view using the `IntersectionObserver` API for a high-end "magazine" feel.
- **Responsive Navigation:** A unique glassmorphism navbar that adapts to scroll depth.
- **Interactive Contact Form:** A custom-built form that validates user input and generates a formatted WhatsApp message.
- **SEO Optimized:** Structured with semantic HTML5 tags and meta-data for better search engine indexing.
- **Infinite Marquee:** A smooth, auto-scrolling brand bar for announcements or featured collections.

---

## 🛠 Tech Stack
- **HTML5:** Semantic structure for accessibility and SEO.
- **Tailwind CSS:** For rapid, custom utility-first styling (No custom CSS files used where avoidable).
- **JavaScript (ES6+):** Handling all DOM manipulations, scroll logic, and API formatting.
- **FontAwesome:** For delicate, high-quality iconography.
- **Google Fonts:** Custom typography integration.
